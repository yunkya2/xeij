1. System disk of Human68k version 3.02 Human68k version 3.02 のシステムディスク

This is the system disk of Human68k version 3.02 distributed for free to the public. This example is for people who want to see the command prompt first or run X-BASIC. 無償公開された Human68k version 3.02 のシステムディスク です。 とりあえずコマンドプロンプトが表示されるところを見たい人と X-BASIC を動かしてみたい人向け。

1.1. Command line コマンドライン 

>java -jar XEiJ.jar -boot=HUMAN302.XDF -config=default -fd1=SXWIN315.XDF -model=Hybrid

2. OTOKO DAMA 男弾（オトコダマ）‐OTOKO DAMA‐

OTOKO DAMA is a vertical scrolling shooting game by Mitsuky. The game screen will be filled by great number of sprites beyond the limit of hardware. A green-colored screen of DSHELL appears first. Then click the leftmost "◎" of "◎遊ぶ（高クロック機用）" to run the game. Press 2/4/6/8 or arrow keys to move the own machine character. Use X key to shoot and Z key to bomb. You can use the automatic shooting function by lighting LED of the hiragana key (" ひら ") which is in bottom left of the standard keyboard. Mitsuky さん による縦スクロールシューティングゲーム 男弾（オトコダマ） です。DSHELL のメニューで「◎遊ぶ（10MHz 機用）」と「◎遊ぶ（高クロック機用）」を選べるようになっているので、速いマシンを持っている人は是非高クロック機用を試してみてください。 スプライト管理システム XSP によって繰り出されるハードウェアの限界を超えた数のスプライトが画面狭しと飛び散ります。 操作はテンキーの 2、4、6、8 またはカーソルキーで移動、X がショット、Z がボンバーです。 標準キーボードの左下にあるひらがなキーの LED を点灯させておくとショットが自動連射になります。

2.1. Command line コマンドライン 

>java -jar XEiJ.jar -boot=OTOKO100.2HD -config=default -model=Hybrid

3. Jong じょんぐ

Jong is a Mahjong game by Toshiyuki Murata who is known for his book "X68000 machine language programming". Play with mouse. There is no BGM nor sound effect. Right click to yell out "Pong", "Chow", "Kong" or "Mahjong". 『X68000 マシン語プログラミング』でおなじみの 村田敏幸さん による麻雀ソフト じょんぐ です。 マウスで遊びます。 音はありません。 ポン、チー、カン、ロンするときは右クリック。

  • Explanation of Jong じょんぐの説明
  • Command line reference of Jong じょんぐのコマンドラインリファレンス
  • Explanation of the configuration file of Jong じょんぐのコンフィギュレーションファイルの説明

3.1. Command line コマンドライン 

>java -jar XEiJ.jar -boot=jong.xdf -config=default -model=Hybrid

4. Gorgeous Reversi ごーじゃすリバーシ

Gorgeous Reversi is a simple Reversi game of my own programming. Disks turning over are animation-displayed on the graphic screen of 512x512 dots and 65536 colors. There is no BGM nor sound effect. 拙作の ごーじゃすリバーシ です。 ボードの上で石が裏返る様子を 512x512 ドット 65536 色モードのグラフィック画面でアニメーション表示させるという無茶なことをしていますが、それを除けばただのリバーシです。 音はありません。

  • Explanation of Gorgeous Reversi ごーじゃすリバーシの説明

4.1. Command line コマンドライン 

>java -jar XEiJ.jar -boot=greversi.hdf -config=default -model=EXPERT
